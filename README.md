Automatically build and publish upstream builds of the Azure/bicep Extension.

## Overview

This project is an automatic build mirroring system for the [Azure/bicep](https://github.com/Azure/bicep) extension. It automatically builds new versions of the upstream repository and publishes them as releases, making them available for various environments and use cases.
